## How to Contribute
We need designer, tester and contributers. **OpenConvert** is prototype. If you want to contribute read [this](./CONTRIBUTING.md)
### Install dependencies

```
$ pnpm install
```

### Run app

```
$ pnpm run start
```
### Run dev server
```
$ pnpm run dev
```
### Package app

To build for windows:

```
$ pnpm run build:win
```
To build for linux:
```
$ pnpm run build:linux
```
To build for mac:
```
$ pnpm build:mac
```
